# ADDITIONALLY

#### Get the Entity ID
Returns the unique identifier of the entity
```vb
entity id of %entity%
```
```vb
%entity%'s entity id
```

#### Get or change the player's Skin value
```vb
skin value of %player%   
```
```vb
%player%'s skin value
```

#### Get or change a player's Skin signature
```vb
skin signature of %player%   
```
```vb
%player%'s skin signature
```
